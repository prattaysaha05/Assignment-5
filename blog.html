Question-3: What is event delegation in the DOM?
By adding an event listener to a parent element rather than to each individual child element, event delegation is a DOM technique that enables the parent to manage events for dynamically introduced child elements as well.
Question-4: What is event bubbling in the DOM?
By starting at the target element and propagating upward to its parent elements, event bubbling is a DOM mechanism that enables each parent element to handle the event unless it is stopped.
